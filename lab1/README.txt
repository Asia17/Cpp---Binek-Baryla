Reverse String - wyświetlanie napisu wspak, ako parametr funkcja  przyjmuje tablicę znaków i wyświetlała ją wspak. Funkcja jest rekurencyjna

Palindrome - funkcję palindrom, sprawdzającą czy podany jako parametr napis jest palindromem. Funkcja  zwraca true gdy napis jest palindromem, a false gdy nie jest

Multiplication table - funkcję która  przyjmuje tablicę dwuwymiarową o rozmiarach 10 na 10 i  uzupełnia ją iloczynami wierszy i kolumn, tworząc tabliczkę mnożenia.

Factorial -  program obliczający silnię
